# IELTS 7.0 雅思备考网站 — 构建完成

## 🎯 项目概况
纯前端静态雅思备考网站，目标用户一年内冲刺 IELTS 7.0 分。
灵感来自「不背单词」app 的交互风格。

## 📁 项目位置
`C:\Users\29369\WorkBuddy\2026-06-08-19-42-44\ielts7/`

## 📐 技术架构
- **纯前端**: HTML5 + CSS3 + Vanilla JavaScript (ES6+)
- **存储**: localStorage (进度/错题/打卡)
- **部署**: GitHub Pages 就绪
- **发音**: Web Speech API (浏览器原生 TTS)
- **路由**: 自建 SPA 路由系统

## 🧩 五大模块
1. **📖 单词背诵** — 不背单词风格：展示→思考→揭示→选择，SM-2 间隔重复算法
2. **🎧 听力训练** — 嵌入式 YouTube 视频，字幕/理解题/听写三种模式
3. **🗣️ 口语练习** — Part1-3 题库，计时训练 + 范文参考 + 关键词
4. **📚 阅读理解** — 分屏阅读 + 自动批改 + 解析
5. **✍️ 写作训练** — 思路框架 + 话题词汇 + 高分范文 + 自评清单

## ⚙️ 核心功能
- ✅ SRS 间隔重复系统（SM-2 改良版）
- ✅ 每日推荐 50 词（新词 + 复习混合）
- ✅ 打卡系统 + 连续天数追踪
- ✅ 错题本/复习模式
- ✅ 暗色/亮色主题切换
- ✅ 响应式设计（手机+平板+桌面）
- ✅ 键盘快捷键（1=首页,2=单词,3=听力,4=口语,5=阅读,6=写作）
- ✅ 每日雅思学习贴士
- ✅ 学习统计仪表盘

## 🚀 启动方式
用任意 HTTP 服务器访问项目目录即可：
```bash
cd ielts7 && python -m http.server 3000
```
当前已在 `http://localhost:3000` 运行预览。
