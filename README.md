## 🧩 五大模块

- **📖 单词背诵** — 不背单词风格，艾宾浩斯遗忘曲线自动复习
- **🎧 听力训练** — 嵌入式视频 + 字幕 + 理解题 + 听写
- **🗣️ 口语练习** — Part1/2/3 题库，计时训练 + 范文
- **📚 阅读理解** — 分屏阅读 + 自动批改
- **✍️ 写作训练** — Task1/2 思路框架 + 高分范文

## ⚙️ 核心功能

- 🧠 艾宾浩斯遗忘曲线（1→2→4→7→15→30→60→120天）
- 📅 每日打卡 + 连续天数追踪
- 🎛️ 多词库选择 + 每日词数自定义（10~200）
- 🖥️ 响应式设计（手机/平板/桌面）
- 🌙 暗色/亮色主题切换

## 🚀 部署

### GitHub Pages（推荐）
1. 仓库 Settings → Pages
2. Source 选 `main` 分支 → Save
3. 访问 `https://maoshaoxing.github.io/-4-6-/`

### 本地运行
```bash
cd ielts7
python -m http.server 3000
```
