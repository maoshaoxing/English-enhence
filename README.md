- 🧠 **艾宾浩斯科学复习** — 1→2→4→7→15→30→60→120 天
- 🎛️ **多词库选择** — 自由组合学习词源
- 📅 **打卡系统** — 连续天数追踪
- 🎨 **暗色/亮色主题** — 一键切换
- 📱 **响应式设计** — 手机/平板/桌面
- 🔥 **每日 100 词** — 可自定义 10~200
